# First
**Ввод:** *два целых числа h и m* — положение часовой стрелки и положение минутной стрелки в отражённом циферблате соответственно
**Вывод:** *два целых числа x и y* — реальное значение текущего времени на часах

### Код программы:
```C++
#include <iostream>
using namespace std;

int main()
{
    int h, m, x, y;
    cin >> h >> m;

    if (h != 0 && h!=6) {
        x = 12 - h;
    }else if (h==0){
        x = 0;
    }else {
        x = 6;
    }

    if (m != 0 && m!=30) {
        y = 60 - m;
    }else if (m==0) {
        y = 0;
    }else {
        y = 30;
    }
    
    cout << x << " " << y;

    return 0;
}

```
